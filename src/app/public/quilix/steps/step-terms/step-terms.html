Quilix values your privacy. When using the local workspace mode (Quilix Workspace), all your data remains on your device and is never transmitted to external servers. Local Storage

Your workspace data is stored using browser IndexedDB or the File System Access API (if enabled). Clearing browser data or revoking file system permissions may result in data loss. We recommend regular backups using the built-in export functionality.
